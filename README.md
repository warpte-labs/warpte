# warpte.com

Landing page for **Warp** — Grok sidebar agent for VS Code & Cursor.

## Stack

- Static site on **Vercel** (`warpte.com`)
- Optional `/api` health endpoint

## Local

Open `index.html` or:

```bash
npx vercel dev
```

## Deploy

```bash
npx vercel --prod
```

## Content

- Header (980px content width)
- Animated pixel **W** (same hero effect as the extension)
- Install command + Install button

Design mockups (20 variants) live in the extension monorepo under `localtools/warp_landing_20_designs.html` until a design is chosen.

## Extension

The VS Code / Cursor extension lives at **https://github.com/warpte-labs/warp** (org: [warpte-labs](https://github.com/warpte-labs)).
